How to run locally:
1. Compile:

```powershell
javac pokemonmanager.java
```

2. Run:

```powershell
java PokemonManager
```

Notes:
- This project is a demo; database credentials and API usage are placeholders.
- Files added for Pages live in the repository and are safe for public browsing.
